Bump Chart

Effective for showing changing rankings across multiple dates. For large datasets, consider grouping lines using colour.

Some useful settings in the index.js

  • columns – set to 'true' will display background columns for each time period (e.g. year)
  • markers – set to 'true' will display dots at the ends of each line
  • highlightNames – put the column names of the data you want to highlight, all other series will be muted

Please make sure than the blue line is on top as this is the primary line colour
