Small multiple grouped column chart

The standard way to compare the size of things. Must always start at 0 on the axis.

Some useful settings in the index.js

  • yMin and yMax – use these to set the lower and upper values of the y-axis
  • numTicksy – set the number of scale lines on the y-axis
  • interval – options available are "century","jubilee" (50 years), decade", "lustrum" (5 years), "years","months","days"
  • endTicks – set to true to show just first and last date on x-Axis
  • fullYear – set to true to show full years for dates on x-Axis
  • dataDivisor – divides data values to more manageable numbers, default is 1
  • hideAxisLabels – set to true to hide axis labels on middle columns of charts to avoid duplication

Setting the rows and columns for each frame

Change these values in the .extend() function for each frame

  • numberOfColumns – set the number of columns
  • numberOfRows – set the number of rows
