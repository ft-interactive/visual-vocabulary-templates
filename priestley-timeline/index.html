Priestley timeline chart

Great when date and duration are key elements of the story in the data

Some useful settings in the index.js

  • interval – options available are "century","jubilee" (50 years), decade", "lustrum" (5 years), "years","months","days"
  • sort – use this to sort the columns into order, your options are – 'ascending', 'descending'
  • sortOn – use this to sort the column groups into order – specify the column number to sort on (ignore name column)
  • showRects – set to true to make your Priestley timeline like a bar chart
  • showMarkers – set to true to make your Priestley timeline like a dot plot
  • showLines – set to true to connect markers
