Grouped symbol chart

Good for comparing numbers across groups, they work best when used on whole numbers and work well in multiple layout form/p>

Some useful settings in the index.js

  • yMin and yMax – use these to set the lower and upper values of the y-axis
  • numTicksy – set the number of scale lines on the y-axis
  • interval – options available are "century","jubilee" (50 years), decade", "lustrum" (5 years), "years","months","days"
  • sort – use this to sort the columns into order, your options are – 'ascending', 'descending'
  • sortOn – use this to sort the column groups into order – specify the column number to sort on (ignore name column)
  • showNumberLabels – set to true to show totals under group category
  • numberOfRows – this number sets the number of rows of icons in each group
  • divisor – use this number to set the unit for each icon. eg. if your data is in millions and you want each icon to represent 250,000, use 0.25 as your divisor
