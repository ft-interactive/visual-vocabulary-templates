Chord diagram

A complex but powerful diagram which can illustrate 2-way flows (and net winner) in a matrix.

Some useful settings in the index.js

  • divisor – use this to set the tick divisions
  • showLabels – turn on/off group labels

Very important

If you have zeroes in your data the geometry can get a bit messy in Illustrator. It may need to some fiddling. Alternatively you can try using Inkscape to output the final graphic
