Stacked bar chart

A simple way of showing part-to-whole relationships but can be difficult to read with more than a few components.

Some useful settings in the index.js

  • xMin and xMax – use these to set the lower and upper values of the x-axis
  • numTicks – set the number of scale lines on the x-axis
  • sort – sets the order of the bars – options available are "ascending","descending", "alphabetical"
