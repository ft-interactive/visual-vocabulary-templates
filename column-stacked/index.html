Stacked column chart

A simple way of showing part-to-whole relationships but can be difficult to read with more than a few components.

Some useful settings in the index.js

  • yMin and yMax – use these to set the lower and upper values of the y-axis
  • numTicksy – set the number of scale lines on the y-axis
  • sort – use this to sort the columns into order, your options are – 'ascending', 'descending', 'alphabetical'
