Fed rate projections beeswarm plot

A quick way of creating a Fed rate projections chart using data from the Federal Reserve. HTML data tables can be found under projection materials on the Federal Reserve Federal Open Market Committee meeting calendar webpage.

Some useful settings in the index.js

  • yMin and yMax – use these to set the lower and upper values of the y-axis
  • numTicksy – set the number of scale lines on the y-axis

Column headings in the csv

  • date – use this column to specify which date the projections were released (this should correspond to a FOMC minutes release). Date will be read in as a string (as if it were a category)
  • Midpoint of target range or target level (Percent) – use this column to specify a projected Fed funds rate
  • Various year columms – use these columns to specify the number of participants who project a particular target level

Please make sure than the blue line is on top as this is the primary line colour
