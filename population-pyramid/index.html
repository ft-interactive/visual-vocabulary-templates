Pyramid chart

A standard way for showing the age and sex breakdown of a population distribution; effectively, back to back histograms

Some useful settings in the index.js

  • xMin and xMax – use these to set the lower and upper values of the x-axis
  • numTicks – set the number of scale lines on the y-axis
  • catLabel – label your bar categories
  • showNumberLabels – set to true to show values at the end of the bars and remove x-axis
