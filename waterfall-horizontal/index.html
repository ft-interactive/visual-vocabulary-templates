Waterfall chart

Designed to show the sequencing of data through a flow process, typically budgets. Can include +/- components

Some useful settings in the index.js

  • yMin and yMax – use these to set the lower and upper values of the y-axis
  • numTicksy – set the number of scale lines on the y-axis
  • showNumberLabels – set to true to show values at the end of the columns
  • total – set to true to show net total bar at the end
