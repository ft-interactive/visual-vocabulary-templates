Calendar heatmap

A great way of showing temporal patterns (daily, weekly, monthly) – at the expense of showing precision in quantity.

Some useful settings in the index.js

  • fiscal – set this to true to use financial years
  • clipYear – set true to clip to year in calendar outline
  • scaleBreaks – gives this an array of breaks for the threshold scale (e.g. [20, 40, 60, 80, 100])
  • scaleType – set colours used in the scale with 'sequentialBlue' or 'sequentialRed'
